bgfx (latest, via github download and following procedure:)
1. mkdir -p 3rdparty_build/bgfx
2. cd 3rdparty_build/bgfx
3. git clone https://github.com/bkaradzic/bx.git --depth 10
4. git clone https://github.com/bkaradzic/bimg.git --depth 10
5. git clone https://github.com/bkaradzic/bgfx.git --depth 10
6. cd bgfx
7. make linux-debug64
8. mkdir ../../../3rdparty/bgfx
9. cp -R include ../../../3rdparty/bgfx/
A. cp -R ../bx/include/bx ../../../3rdparty/bgfx/include/
B. cp -R .build/linux64_gcc/bin ../../../3rdparty/bgfx/

SDL2 (2.28.4)
1. cd 3rdparty_build
2. git clone https://github.com/libsdl-org/SDL.git -b release-2.28.4 --depth 1
3. cd SDL
4. mkdir build
5. cd build
6. ../configure --enable-shared=no prefix=$PWD/../../../3rdparty/sdl2
7. make -j8
8. make install
9. make clean

Dear ImGui (1.89.9 Docking)
Dear ImGui is a git submodule of this repository.
So you should use "git submodule init" and "git submodule update" to fill the src folder.
